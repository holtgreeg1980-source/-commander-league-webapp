# Commander League Rules

## Deck construction
- Each deck file must contain exactly 100 total cards by quantity.
- One of those cards is marked as the commander.
- At game start, the commander is removed from the deck file and placed in the command zone.
- The remaining 99 cards are shuffled into the library.
- Opening hands are 7 random cards drawn from the shuffled 99-card library.
- The commander is never drawn from the library.

## Commander casting
- Commanders may be cast from the command zone.
- Each time a commander is cast from the command zone after the first, it costs 2 more generic mana for each previous command-zone cast.
- If a commander would go to graveyard or exile, its owner may move it back to the command zone.

## League setup
- Season 1 starts with four decks: Eldrazi Incursion, Pantlaza, Aesi, and Disa.
- More decks can be added in Deck Manager using the same 99+1 format.

## Randomization
- Each game uses a random seed.
- The app shuffles each 99-card library and draws real opening hands.
- Draws come from the top of the shuffled library, so cards cannot appear twice unless legally moved.

## Tracking
The app tracks library, hand, battlefield, graveyard, exile, command zone, life, commander tax, commander damage, tokens, and game log.
